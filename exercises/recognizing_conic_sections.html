這個圖形是一個對稱的圓環形狀；在線上的每一個點離中心點的距離都一模一樣。

這個圖形要是一個圓，才會有這些特性。

雙曲線

這個圖形是由兩個漸進線所構成，而兩邊各是 U 的形狀 。

plot( function( x ) { return (B/A) * x; }, [-10, 10], { stroke: "gray" } ); plot( function( x ) { return (B/A) * -1 * x; }, [-10, 10], { stroke: "gray" } );

因此，這個圓錐曲線叫做雙曲線。
