在等式的左邊, c 先加上 d 後, 再乘上 b.

在等式的右邊, 我們分別將 b 乘上 c 和 d , 再將他們的乘積加起來.

A+B(C+D)=A+(B)(C)+(B)(D) 代表了那個特性:

  • value

在等式的左邊, B 乘上 C 加上 D的和. 在等式的右邊, 我們先分別將 B 乘上 C 和 D 再將他們的乘積加起來.

求左邊的值為:

A+B(C+D)=DISTRIBUTIVE_RESULT

求右邊的值, 而答案的一樣的:

A+(B)(C)+(B)(D)=DISTRIBUTIVE_RESULT

乘法的運算中，可以將被乘數B各分配給括弧中C和D相加的數字, 而這被叫作分配律

右邊哪一個算式是加法分配律?

a+b(c+d)=a+bc+bd

  • value

在正確的答案中, 在等式的左邊, 我們先得到c加上d的和, 再乘上b.

在等式的右邊, 我們先分別將b 乘上 c 和 d, 再將他們的乘積加起來.

在這個等式中, 我們將b 分配 給 c 和 d, 所以我們稱這個為分配律.

a+b(c+d)=a+bc+bd 就表示了分配律.
